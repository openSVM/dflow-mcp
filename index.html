Quick Start

Claude Desktop

Add to Claude Desktop configuration:

{ "mcpServers": { "dflow-mcp": { "command": "bun", "args": ["run", "./src/index.ts"] } } }

Terminal Usage

Run locally for development:

git clone https://github.com/openSVM/dflow-mcp.git cd dflow-mcp bun install bun run dev
