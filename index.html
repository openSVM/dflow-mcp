Quick Start EASY SETUP

Already deployed and ready to use! No installation needed.

WEB MCP ENDPOINT
https://dflow.opensvm.com/api/mcp
Use this URL directly in any MCP client that supports HTTP endpoints
{ "mcpServers": { "dflow-mcp": { "url": "https://dflow.opensvm.com/api/mcp" } } }

Zero setup required • Production-ready • Always online • Try the playground below!

Install locally via Smithery for offline usage

npx @smithery/cli install dflow-mcp-server --client claude

This automatically configures the server for Claude Desktop. Restart Claude Desktop and you're done!

Clone the repository and run locally for development

git clone https://github.com/openSVM/dflow-mcp.git cd dflow-mcp bun install bun run dev

For local development and testing. Or just use the hosted version above!

Manual stdio configuration (if you prefer local over hosted)

{ "mcpServers": { "dflow-mcp": { "command": "bun", "args": ["run", "/path/to/dflow-mcp/src/index.ts"] } } }

Replace /path/to/dflow-mcp with your actual installation path. But seriously, just use the hosted endpoint!

Available Tools

Comprehensive access to prediction market data through 24 specialized tools across 8 categories.

get_events

Get paginated list of all prediction market events with optional filtering

get_event

Get detailed information about a single event by ticker with nested markets

search_events

Search events by title or ticker with sorting and pagination support

get_markets

Get paginated list of all markets with volume and liquidity data

get_market

Get detailed market information by ticker symbol

get_market_by_mint

Retrieve market details using Solana mint address

get_markets_batch

Fetch multiple markets in a single request (up to 100)

get_trades

Get trade history across all markets with filtering

get_trades_by_mint

Get trade history for a specific market by mint address

get_forecast_percentile_history

Historical forecast percentile data for time series analysis

get_forecast_percentile_history_by_mint

Forecast history by mint address with customizable time ranges

get_event_candlesticks

OHLC candlestick data for event-level price action

get_market_candlesticks

Market-level candlestick data for technical analysis

get_market_candlesticks_by_mint

Candlestick data retrieval using mint addresses

get_live_data

Real-time milestone information and live data feeds

get_live_data_by_event

Live data for specific events and milestones

get_live_data_by_mint

Real-time data retrieval by mint address

get_series

Get all series templates and category definitions

get_series_by_ticker

Retrieve specific series information by ticker

get_outcome_mints

Get outcome mint addresses for prediction markets

filter_outcome_mints

Filter addresses to identify outcome mints

get_tags_by_categories

Category-tag mapping for market organization

get_filters_by_sports

Sports-specific filtering options and metadata

candlestick_chart

Generate ASCII candlestick charts for visualization
